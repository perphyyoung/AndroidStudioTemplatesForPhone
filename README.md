# Android Studio Templates For Phone
This repository includes several templates available for phone, provided by Android Studio.

### List as following
1. Empty Activity (main)
* Basic Activity
* Fullscreen Activity
* Master/Detail Flow Activity
* Login Activity
* Navigation Drawer Activity
* Scrolling Activity
* Settings Activity
* Tabbed Activity
